## Context

Отчет по ПМ 01 предназначен для демонстрации результатов преддипломной практики студента Колледжа АлтГУ. Разработанный программный продукт Testosteron представляет собой веб-приложение для проведения тестирования на платформе ASP.NET Core MVC.

## Goals / Non-Goals

**Goals:**
- Создать структурированный отчет по ПМ 01
- Описать теоретические основы разработки информационных систем
- Изложить принципы тестирования программного обеспечения
- Представить практическую часть: проектирование, разработку, тестирование системы Testosteron
- Составить руководство пользователя

**Non-Goals:**
- Изменение исходного кода проекта
- Добавление новых функций в систему
- Развертывание системы на production

## Decisions

1. **Формат отчета**: Markdown с последующей конвертацией в Word/PDF
2. **Структура отчета**: Согласно методическим рекомендациям:
   - Введение (актуальность, цель, задачи)
   - Глава 1: Об организации
     - 1.1 Характеристика организации
     - 1.2 Основы разработки информационных систем
     - 1.3 Принципы тестирования программ
   - Глава 2: Виды выполненных работ
     - 2.1 Проектирование системы
     - 2.2 Разработка ИС
     - 2.3 Тестирование
     - 2.4 Документация
   - Заключение
   - Список источников

3. **Организация**: Колледж АлтГУ (ФГБОУ ВО Алтайский государственный университет)
4. **Объект исследования**: Система Testosteron
5. **Предмет**: Процесс разработки и тестирования информационной системы

## Risks / Trade-offs

- [Риск] Недостаток информации о структуре организации → Использовать открытые данные с сайта вуза
- [Риск] Сложность создания UML диаграмм → Использовать PlantUML или аналоги
- [Риск] Объем отчета → Следовать рекомендациям (20-25 страниц)
