## Why

Студентам Колледжа АлтГУ требуется создать отчет по профессиональному модулю ПМ 01 для прохождения преддипломной практики. Отчет должен соответствовать методическим рекомендациям и включать описание разработанного программного продукта Testosteron - системы тестирования.

## What Changes

- Создание структурированного отчета по ПМ 01
- Описание организации (Колледж АлтГУ)
- Теоретическая часть: основы разработки информационных систем и принципы тестирования
- Практическая часть: описание проектирования, разработки и тестирования системы Testosteron
- Составление документации и руководства пользователя

## Capabilities

### New Capabilities
- pm-01-report: Создание отчета по ПМ 01 для Колледжа АлтГУ
