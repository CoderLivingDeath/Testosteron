---

## 1. Введение

Настоящий документ содержит результаты модульного (unit) тестирования системы онлайн-тестирования Testosteron. Тестирование проведено с использованием фреймворка MSTest и библиотеки Moq для мокирования зависимостей.

**Цели тестирования:**
- Проверка корректности работы паттерна Result<T>
- Проверка бизнес-логики TestManager
- Проверка функциональности репозиториев
