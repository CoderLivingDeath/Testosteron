# Testosteron

Веб-приложение для проведения тестирования на платформе ASP.NET Core MVC.

## Описание

**Testosteron** — это система тестирования, разработанная для образовательных учреждений. Приложение позволяет создавать тесты с различными типами вопросов, проходить тестирование и просматривать результаты.

## Возможности

- **Управление тестами**: создание, редактирование и удаление тестов
- **Типы вопросов**:
  - Текстовый ответ
  - Выбор одного варианта
  - Выбор нескольких вариантов
- **Аутентификация и авторизация**: система учётных записей с разделением ролей
- **Административная панель**: управление тестами и пользователями
- **Адаптивный интерфейс**: корректная работа на мобильных устройствах

## Технологический стек

- **Backend**: ASP.NET Core 8.0 (MVC)
- **База данных**: PostgreSQL / SQLite (для разработки)
- **ORM**: Entity Framework Core 8.0
- **Frontend**: Bootstrap 5, Razor Pages
- **Аутентификация**: ASP.NET Core Identity

## Требования

- .NET 8.0 SDK
- PostgreSQL 14+ (или SQLite для разработки)

## Установка и запуск

### Клонирование репозитория

```bash
git clone https://github.com/CoderLivingDeath/Testosteron.git
cd Testosteron
```

### Сборка проекта

```bash
dotnet build
```

### Запуск

```bash
dotnet run --project Testosteron
```

Приложение будет доступно по адресу: `https://localhost:7000`

### Учётная запись администратора

При первом запуске автоматически создаётся администратор:

- **Логин**: admin
- **Email**: my@email.com
- **Пароль**: Superpassword

## Структура проекта

```
Testosteron/
├── Controllers/          # Контроллеры MVC
├── Domain/              # Доменная логика
│   ├── Entities/       # Сущности БД
│   ├── Repositories/   # Репозитории
│   └── ApplicationDbContext.cs
├── Services/            # Бизнес-логика
├── Models/              # Модели представления
├── Views/               # Представления Razor
├── Areas/               # Области (Admin)
└── wwwroot/            # Статические файлы
```

## База данных

### Основные сущности

- **Test**: тест (Id, Title, Description, TestFields)
- **TestField**: вопрос (Title, Type, Required, Options)
- **Answers**: ответы пользователя (TestId, UserId, Content)
- **ApplicationUser**: пользователь системы

## Тестирование

Проект включает unit-тесты в `Testosteron.tests/`.

Запуск тестов:

```bash
dotnet test
```
